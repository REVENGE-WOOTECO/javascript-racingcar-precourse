# 구현 기능 목록

## 1. 자동차 이름 입력 받기

- [x] 자동차 이름은 쉼표를 기준으로 구분
- [x] 이름 확인
  - [x] 이름 사이 공백 불허용
    - [x] 공백이 있다면 alert "공백 없이 입력해주세요."
  - 1글자 미만, 5글자 초과:
    - [x] alert으로 "자동차 이름은 1자 이상, 5자 이하여야 합니다. 예) 12345 (허용), 123456 (불가)"
  - 5글자 이하:
    - [x] 이름을 배열로 return
  - 이름 중복:
    - [x] alert "중복되지 않은 이름을 사용해주세요."
    - [x] 자동차 이름이 1개 미만이면 alert "자동차 이름을 2개 이상 입력해주세요."
- [x] 자동차 이름을 배열로 return
- [x] 정확한 이름을 입력 받으면 게임 횟수 입력할 수 있게 함

## 2. 게임 횟수 입력 받기

- [ ] 입력이 0이거나 음수일 경우:
  - [ ] alert "1이상의 수를 입력해주세요."
- [ ] 입력이 1이상의 수일 경우:
  - [ ] 게임 횟수 return

## 3. 무작위 값 구하기

- [ ] 0 ~ 9 사이의 무작위 값을 return
- [ ] return 값에 따라 전진 또는 멈춤할지 판단 
  - [ ] 값 >= 4 일 때 return +1
  - [ ] 값 < 4 일 때 return 0
- [ ] 위의 행동을 자동차 개수만큼 반복

## 4. 결과 출력하기

- [ ] "자동차 이름: " 의 형태로, 전진할 때마다 끝에 "-"를 추가
- [ ] 한 사이클이 끝날 때마다 empty line 출력

## 5. 게임 결과  

- [ ] 모든 결과 출력되면, 최종 우승자를 "최종 우승자: (이름)"의 형태로 출력 
  - 우승자 한명 이상일 수도 있음
